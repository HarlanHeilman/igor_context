Making Boxes and Violins Look Different
At times you may want one of the boxes or violins in a trace to have a different appearance to distinguish 
the corresponding dataset from others in the trace. The Modify Box Plot and Modify Violin Plot dialogs 
support this by allowing you to select a single dataset from a trace.
This graphic shows the Modify Box Plot dialog with a box plot trace made with a list of three waves. The 
first wave in the list was wave0, so the name of the trace is "wave0". Clicking the triangle icon for the wave0 
trace discloses a list of the three datasets, one for each wave.
If the top wave0 item were selected, changes made in the dialog would apply to all datasets in the trace. 
Here we have selected the second dataset corresponding to wave1 so changes made in the dialog apply only 
to the dataset and consequently to the second box in the box plot.
